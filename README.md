## ccgo-template

template used to init an ccgo cross-platform project.

## Usage

```
# ensure ccgo installed
pip3 install ccgo

# create lib
ccgo lib create <YOUR_PROJECT_DIR> --template-url=https://github.com/zhlinh/ccgo-template.git
```
